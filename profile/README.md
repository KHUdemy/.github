## 📖 주제

* **KHUdemy**: 생성형 AI를 이용한 자동화 교육 미디어

<br>

## 🌼 소개

* "KHUDemy"는 **AI 기반 학습 플랫폼**으로, 영상을 활용한 효과적인 학습을 지원합니다.

* 현재 학습 시장에서 **영상 학습에 대한 수요가 증가**하고 있지만, 이를 관리하고 성취도를 측정하는 데에는 많은 어려움이 있습니다. KHUdemy는 이러한 문제를 해결하기 위해, **칸반 보드를 통해 시각적인 일정 관리를 제공**함과 동시에, **AI 기술을 활용하여 학습 계획의 유연성을 제공**합니다. AI 기술은 GPT-4와 DALL-E 3를 활용하여 영상 자막 기반의 이미지 퀴즈 자동 생성을 통해 개인 맞춤형 학습 경험을 제공합니다.

* 칸반 보드를 통해 학습자는 본인의 진척도를 쉽게 파악하고 일정을 조율할 수 있습니다. 또한, **GPT-4는 영상 자막을 분석하여 자동으로 퀴즈를 생성**하고, **DALL-E 3를 이용하여 퀴즈를 시각적으로 흥미롭게 만들어** 학습 성취도를 높일 수 있습니다.

* Google API와 YoutubeTranscript API를 활용하여 영상 자막 및 데이터를 수집하고, AI 모델을 이용해서 개인화된 학습 콘텐츠를 제작합니다. 이를 통해 학습자들에게 자신의 학습 과정을 조절하며 효율적으로 성장할 수 있는 환경을 마련하고자 합니다.

<br>

## 🛰️ 기술

* AI: `DALL-E 3`, `GPT-4`

* Cloud Service: `Amazon S3`, `Amazon EC2`, `Amazon RDS`

* Backend: `Sprint Boot`, `Nginx`

* Frontend: `React`
